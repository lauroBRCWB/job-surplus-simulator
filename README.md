# Job surplus simulator
*This app explores a core macroeconomic question: job displacement vs AI job productivity surplus.*

If major economies lose 20–50% of their population by 2100, will AI-driven job loss be real — or will automation simply replace missing workers?

It models:
- Population decline to 2100
- Compounded productivity growth (AI + new tech)
- Optional real output growth
- Cross-country comparisons (Germany, France, US, Sweden, Brazil, China)

Instead of forecasting unemployment, it performs a balance-sheet comparison:
- Available workers (after demographic decline)
- Required workers (after productivity gains)

The result shows whether a country faces:
- Surplus labor (displacement pressure), or
- Labor shortage (automation as compensation)

It is not a prediction engine — it is a structural thinking tool.
